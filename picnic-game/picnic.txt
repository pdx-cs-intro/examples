to collect the list of names
    make an empty list of names
    get the count of users from the user
    for each user
        get a name from the user
        save it on the end of the name list
    return the name list

to play a round
    for each name in the name list
        ask for the food
        get the first character of the name
        get the first character of the food
        compare them (ignoring case)
        print the appropriate message for the comparison

collect the list of names
repeatedly
    play a round with the names in the list
    ask whether to keep going
    if not, stop the game
